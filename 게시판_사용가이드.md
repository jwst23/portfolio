# 자유게시판 사용 가이드

본 자유게시판은 데이터베이스(DB) 서버 없이 귀하의 웹 브라우저 로컬 저장소(localStorage)를 활용하여 동작하도록 간략하게 구성되었습니다. 관련 정보, 요청사항, 또는 문의사항을 남기고 답변을 작성할 수 있습니다.

## 주요 기능 및 사용 방법

### 1. 글 작성하기 (문의사항/요청사항 등록)
- **이름**: 작성자 본인의 이름을 입력합니다. (필수)
- **이메일**: 연락 가능한 이메일을 입력합니다. (필수)
- **내용**: 관련 정보나 요청사항 및 문의사항을 자유롭게 입력합니다. (필수)
- 모든 항목을 입력한 뒤 **'등록하기'** 버튼을 클릭하면 새로운 글이 게시판 목록에 추가됩니다.

### 2. 목록 확인하기
- 등록된 게시물은 작성 시간과 함께 게시판 목록(글 작성란 하단)에 최신순으로 표시됩니다.
- 이름, 이메일, 그리고 작성한 내용이 카드 형태로 보여집니다.

### 3. 답변하기 (댓글 달기)
- 각 게시물 우측 하단의 **'답변 달기'** 버튼을 클릭하여 해당 글에 대한 답변을 작성할 수 있습니다.
- 답변을 입력하고 엔터를 누르거나 등록 버튼 모양을 클릭하면 해당 게시물 아래에 답변이 추가됩니다.

### 4. 삭제하기
- 게시물 본인 또는 관리자는 **'삭제'** 버튼을 클릭하여 게시물과 해당 글에 달린 답변들을 삭제할 수 있습니다.

## 유의사항
- 게시판의 데이터는 **현재 사용 중인 브라우저(PC/모바일)** 에만 저장됩니다.
- 다른 컴퓨터나 브라우저, 기기를 사용하거나, 브라우저의 접속 기록(캐시/쿠키/로컬 데이터)을 모두 삭제할 경우 게시판의 내용이 사라지게 됩니다.
- 실제 서비스로 배포하실 경우에는 백엔드 개발 및 데이터베이스 연동이 필요합니다.
